Sous Windows 64 bits :

Version 3.12 : 

Quelques pré-requis pour installer AperodeDenis sans installateur :

0) Installer Micmac (voir le site de l'IGN)

1) Télécharger et installer Python 3.4 sous c:\python34 :
https://www.python.org/downloads/release/python-340/

Choisir "Windows x86-64 MSI installer" qui télécharge l'installateur python-3.4.0.msi.

2) Ouvrir une console (exécuter le programme "cmd" pour ouvrir la console)
modifier le path : 
> set Path=%Path%;C:\Python34\Scripts;c:\python34

3) Dans la console Installer Pillow :
> pip install pillow

4) Copier dans un répertoire 4 les éléments nécessaires à AperoDeDenis :

AperoDeDenis.py
logoCerema.jpg
logoIgn.jpg
locale (toute l'arborescence)

5) Lancer AperoDeDenis en double cliquant sur aperoDedenis.py.
Utiliser le menu paramètrage pour Associer le répertoire bin de Micmac.

Vous pouvez consulter le menu "Aide" et le document "Notice Installation interface graphique micMac version 3.pdf".

L'installation d'un outil de visualisation des nuages de points 3D est conseillé : CloudCompare ou Meshlab.
