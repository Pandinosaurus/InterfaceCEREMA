Sous Windows 64 et 32 bits :

Des installateurs sont disponibles pour la version 3.33.

Pour installer la version 3.33 : utiliser les installateurs 32 et 64 bits.

La version 4 fixe 2 régressions de la version 3.2 et ajoute le calcul d'indices surfaciques.
Il n'y a pas d'installateur pour la version 4. 

Pour installer la Version 4 sous Windows : 

Quelques pré-requis pour installer AperodeDenis sans installateur :

0) Installer Micmac (voir le site de l'IGN).
   Installer CloudCompare pour visualiser les fichier ply.

1) Télécharger et installer Python 3.6 (minimum 3.4) sous c:\python3 :
https://www.python.org/downloads/release/python-360/


2) Ouvrir une console (exécuter le programme "cmd" pour ouvrir la console)
modifier le path : 
> set Path=%Path%;C:\Python34\Scripts;c:\python34

3) Dans la console Installer les 4 modules : pillow, matplotlib, numpy, scipy :
> pip install pillow
> pip install matplotlib
> pip install numpy
> pip install scipy

4) Copier dans un répertoire 4 les éléments nécessaires à AperoDeDenis :

AperoDeDenis.py
logoCerema.jpg
logoIgn.jpg
locale (toute l'arborescence)

5) Lancer AperoDeDenis en double cliquant sur aperoDedenis.py.
Utiliser le menu paramètrage pour Associer le répertoire bin de Micmac et celui de CloudCompare.

Vous pouvez consulter le menu "Aide" et le document "Documentation AperoDeDenis.pdf".

L'installation d'un outil de visualisation des nuages de points 3D est conseillé : CloudCompare ou Meshlab.
