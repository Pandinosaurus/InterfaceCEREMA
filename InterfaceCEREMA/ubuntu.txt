
Version 2.6 : 

quelques pré-requis pour installer sans installateur :
sudo apt-get install python3-tk python3-pil python3-imaging-tk

lancer l'appli
python3 AperoDeDenis.py

le fichier .rpm permet d'installer l'application mais nécessite que python3 soit la version de python par défaut.

L'installation du paquet .rpm peut générer un message "paquet de mauvaise qualité". 
Vous pouvez cependant installer l'interface sans risque.

Version 3.10 sous Ubuntu sans installateur :
les fichiers d'installation .rpm et .deb ne sont pas disponibles.

Il faut recopier le répertoire "locale" dans le même répertoire que AperoDeDenis.py (pour la traduction anglaise) 
Il faut installer python3 et pil comme indiqué dans le pré requis ci-dessus.
