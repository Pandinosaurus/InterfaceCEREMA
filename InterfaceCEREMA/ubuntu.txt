Sous Ubuntu/linux :

Un pré-requis pour installer sans installateur :

sudo apt-get install python3-tk python3-pil python3-imaging-tk

lancer l'appli

python3 AperoDeDenis.py


Version 3 sous Ubuntu  :

Il faut recopier le répertoire "locale" dans le même répertoire que AperoDeDenis.py (pour la traduction anglaise) 
Il faut installer python3 et pil comme indiqué dans le pré requis ci-dessus.

Version 4 :

La version 4 fixe 2 régressions de la version 3.2 et ajoute le calcul d'indices surfaciques.

L'import des modules python suivant est nécessaire : numpy, scipy, matplotlib.
